Free, Easy to Use Web Tools and More! By Oli Everard

Please have a browse around and explore some of my projects, free tools and more! The list is hopefully still growing if I haven't become lazy. Feel free to steal any of the bits you like, if you manage to find something remotely useful on here.

On my 'Tools' page, there are multiple tools for completing tasks such as file type conversion, as well as an email extractor, and an email web scraper powershell script. If you're after more fun, check out the "Useless Projects" tab instead.

My Tools All My Repositories

I have designed all of the tools to use client side scripting where possible, and not upload your data to the web. All of them are available to view and download on my Github as well, so you don't have to use my website to use them if you wish to host your own instead. I decided to put these together because I was fed up of vetting all the online tools I used, to try and figure out if my data was uploaded to the internet.

On the "Useless Projects" page, I will be uploading some of the more fun ideas that come to me. These may be little games, stupid jokes, or something I have built whilst teaching myself something new. I'll try to keep it creative!


If you REALLY like this site, please consider hitting the donate button in the navbar. Or don't, I don't mind.

If you're wondering what this site will become, I plan to focus on expanding the "tools" part of it to become more useful, and to streamline the tasks that I do regularly. I also want to replace some tools I use regularly with my own, and hopefully come up with some new ones. ALl of this will be done in the client side of the browser where it can. This does mean that some tools will take a long while to process: like HEIC2JPG. But, it also means that your data is safe

At current, much of my code is horribly uncommented. I plan to go back and resolve this at some point. Most likely. :p

This site is hosted through Github Pages, so you can see all of the code for everything on my Github.
